wounit by hprange

The WOUnit framework contains a set of utilities for testing WebObjects applications using JUnit 4.7 or later capabilities.

No Database Access Needed: all the logic is handled in memory for fast unit testing and integration testing. Wonderful: developed on top of Wonder classes, make possible the use of the augmented transaction process specified by the ERXEnterpriseObject interface. Easy to use: No extension required for test classes. The WOUnit library make use of the MethodRule approach provided by JUnit 4.7. It also load EOModels automatically.

Dependencies

JUnit 4.7 or later Project Wonder 5 WebObjects 5

Install

Maven users have to add the dependency declaration: com.wounit wounit 1.0 Non Maven users have to: Download the wounit.jar. Add the wounit library to the build path.
